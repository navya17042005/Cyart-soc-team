# Alert Triage with Automation

## Objective
To analyze security alerts and perform automated validation using threat intelligence tools.

## Tools Used
- Wazuh
- VirusTotal
- TheHive (simulated)

## Steps
1. Generated alert in Wazuh
2. Analyzed alert (triage)
3. Validated using VirusTotal
4. Simulated automation using TheHive

## Result
Alert was analyzed and validated successfully. No malicious activity was detected.

## Conclusion
Automation improves efficiency in SOC operations by reducing manual effort and improving accuracy
