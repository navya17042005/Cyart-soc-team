# Evidence Analysis Lab

## Objective
To analyze network connections and maintain chain-of-custody for digital evidence.

## Tools Used
- Velociraptor
- FTK Imager

## Steps Performed
1. Collected system data from Windows VM
2. Executed netstat query using Velociraptor
3. Identified suspicious network connections
4. Documented findings
5. Maintained chain-of-custody

## Key Findings
- Suspicious connection to external IP
- Port 4444 detected (possible reverse shell)
